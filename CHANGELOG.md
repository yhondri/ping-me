# Change Log

All notable changes to the "ping-me" extension will be documented in this file.

## [1.1.0]

- Added `PermissionRequest` hook support via `vscode://yhondri.ping-me/permissionRequest?project=...&path=...`.
- Notification message now supports custom text via `?message=` query param.
- Added `Ping Me: Permission Request` command in the Command Palette.

## [1.0.1]

- Added extension icon.

## [1.0.0]

- Initial release.
